1. Box Model - Modelo de Caixa

Cada elemento HTML é uma "caixa" composta por 4 partes:

MARGIN (margem externa - laranja)
BORDER (borda - azul)
PADDING (preenchimento - verde)
CONTENT
(conteúdo - vermelho)
MARGIN
Espaço externo
margin: 40px;
BORDER
Borda
border: 8px solid;
PADDING
Espaço interno
padding: 20px;
CONTENT
Conteúdo real
width x height
.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid black;
  margin: 15px;
}

2. Margin e Padding - Espaçamentos

Sintaxe Exemplo Significado
1 valor margin: 20px; Todos os lados = 20px
2 valores margin: 10px 20px; Top/Bottom = 10px, Left/Right = 20px
4 valores margin: 10px 20px 30px 40px; Top, Right, Bottom, Left (sentido horário)
Individual margin-top: 15px; Apenas o topo
Centralizar margin: 0 auto; Centraliza horizontalmente
Padding: 20px
Margin: 15px
Margin: 0
Margin: auto
(centralizado)

5. Outline - Contorno Externo

Outline é como uma borda, mas NÃO ocupa espaço no layout.

Outline sólido
outline: 4px solid orange;
outline-offset: 5px;
Outline tracejado
outline: 4px dashed green;
outline-offset: 10px;
Aspecto Border Outline
Ocupa espaço ✅ SIM ❌ NÃO
Afeta layout ✅ SIM ❌ NÃO
border-radius ✅ Aceita ❌ NÃO aceita
Offset (distância) ❌ NÃO tem ✅ TEM (outline-offset)

8. Transparência - Opacity e RGBA

Opacity (afeta TODO o elemento):

100%
(opaco)
75%
50%
25%

RGBA (afeta só a cor):

rgba(28, 114, 147, 0.5)
Background 50% transparente
Texto 100% opaco
rgba(28, 114, 147, 0.9)
Background 90% opaco
Texto 100% opaco
/* Opacity - afeta tudo */
opacity: 0.5; /* 50% transparente */

/* RGBA - afeta só a propriedade */
background: rgba(255, 0, 0, 0.5); /* vermelho 50% */
color: rgba(0, 0, 0, 0.7); /* preto 70% opaco */

10. Position - Posicionamento

static (padrão)
Fluxo normal do documento
relative
top: 20px; left: 20px;
(relativo à posição original)
absolute
top: 50px; right: 50px;
(relativo ao container)

O container tem position: relative para ser a referência do absolute.

/* Relative */
position: relative;
top: 20px; left: 20px;

/* Absolute */
position: absolute;
top: 50px; right: 50px;

/* Fixed (fixo na viewport) */
position: fixed;
bottom: 20px; right: 20px;
↑ Fixo!
