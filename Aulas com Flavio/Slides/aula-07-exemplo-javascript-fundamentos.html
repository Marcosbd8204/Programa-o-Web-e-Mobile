2. Variáveis (let, const)

Palavra-chave Pode Reatribuir? Quando Usar
let ✅ SIM Valores que mudam
const ❌ NÃO Valores constantes (PADRÃO)
var ✅ SIM ❌ NÃO use (legado)

Experimente:

// Usando let (pode mudar) let nome = 'João'; console.log(nome); // 'João' nome = 'Maria'; console.log(nome); // 'Maria' - OK! // Usando const (não pode mudar) const PI = 3.14159; console.log(PI); // 3.14159 // PI = 3.14; // ERRO! Não pode reatribuir const

5. Operadores de Comparação

⚠️ Importante: Use sempre === (igualdade estrita) em vez de ==

Teste Comparações:

// == compara apenas VALOR console.log(5 == '5'); // true ⚠️ // === compara VALOR E TIPO (use sempre!) console.log(5 === '5'); // false ✓ // Outros operadores console.log(10 > 5); // true console.log(10 < 5); // false console.log(10 >= 10); // true console.log(5 !== '5'); // true (diferente em tipo)

6. Operadores Lógicos

&& (E / AND)

Ambos precisam ser true

true && true → true true && false → false false && true → false false && false → false

|| (OU / OR)

Pelo menos um true

true || true → true true || false → true false || true → true false || false → false

! (NÃO / NOT)

Inverte o valor

!true → false !false → true

Exemplo Prático - Verificar Idade:



let idade = 20; let temCarteira = true; // E (AND) - ambos precisam ser true if (idade >= 18 && temCarteira) { console.log('Pode dirigir'); } // OU (OR) - pelo menos um true if (idade < 12 || idade > 65) { console.log('Desconto especial'); } // NÃO (NOT) - inverte if (!temCarteira) { console.log('Não pode dirigir'); }
